API Information

Technical details about the API integration used in PromptLabs.

API Provider

PromptLabs uses the Groq API for fast and reliable AI inference. Groq provides access to state-of-the-art language models with incredibly fast response times.

Endpoint

https://api.groq.com/openai/v1/chat/completions

Model Used

llama-3.3-70b-versatile

Request Format

The API accepts POST requests with the following structure:

{ "model": "llama-3.3-70b-versatile", "messages": [ { "role": "user", "content": "Your prompt here" } ] }

Response Format

The API returns responses in the following format:

{ "choices": [ { "message": { "content": "AI response content" } } ] }
